# INTI-FRAME
Agente de inteligencia artificial
aplicada al desarrollo sostenible.
Actúa como guía turístico,
Una herramienta tecnológica para resolver una problemática económica y social real en el sector turístico,
enfocado directamente en la meta 8.9 de la ONU, que
busca promover el turismo sostenible,
crear puestos de trabajo,
fomentar la cultura y los productos locales.
Resuelve las malas experiencias de los turistas.
